# intro-in-ai-vgg-cifar10

Этот репозиторий содержит рабочий код для обучения модифицированной модели vgg16 для классификации изображений cifar-10. 
Файлы содержат загрузку и предобработку данных, формирование пакетов данных, описание архитектуры модели, запуск обучения с сохранением результатов, финальную оценку точности обученной модели на валидационном наборе данных и вывод метрик и графиков с результатами обучения.
